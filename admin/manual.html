¿Cómo actualizar el contenido del sitio?

Puedes elegir entre estas opciones para gestionar el contenido de tu sitio web:

Opción 1: Editar directamente en GitHub Recomendado

La forma más simple y directa de editar el contenido:

  1. Ve a tu repositorio: sci6377/web-bachillerato-heroes
  2. Navega a la carpeta data/
  3. Haz clic en el archivo que quieres editar (ej: testimonios.json)
  4. Haz clic en el ícono del lápiz para editar
  5. Modifica el contenido en formato JSON
  6. Haz clic en "Commit changes"
  7. ¡Los cambios aparecerán automáticamente en tu sitio web!
Ir a GitHub

Archivos de Contenido

Estos son los archivos que puedes editar para cambiar el contenido:

  • config.json
    Información general de la institución
  • testimonios.json
    Experiencias de egresados
  • noticias.json
    Comunicados y avisos
  • docentes.json
    Información del personal
  • eventos_calendario_pwa.json
    Eventos y fechas importantes
  • avisos_pwa.json
    Avisos para la PWA

Opción 2: GitHub Desktop

Para una experiencia más cómoda:

  1. Descarga GitHub Desktop
  2. Clona tu repositorio
  3. Edita los archivos JSON con cualquier editor de texto
  4. Haz commit y push de los cambios
Descargar GitHub Desktop

Opción 3: Editor JSON Online

Para editar JSON de forma más visual:

  1. Copia el contenido del archivo JSON desde GitHub
  2. Pégalo en un editor JSON online
  3. Edita visualmente
  4. Copia el resultado y pégalo de vuelta en GitHub
Abrir Editor JSON
